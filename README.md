# 🔐 Password Generator & Credential Manager

A simple Python CLI application to generate secure passwords, analyze password strength, and manage saved credentials.

## Features
- Generate secure random passwords
- Analyze password strength
- Save, search, view, and delete credentials
- Store data in a JSON file

## Technologies
- Python 3
- JSON
- Built-in Python modules

## Installation

```bash
git clone https://github.com/itsrahul77/Password-Generator.git
cd Passwordgenerator
python main.py
```
